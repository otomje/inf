Мої захоплення

На цьому сайті я розповім частину того, що мені подобається

Фронтенд

HTML

Знаю досить багато тегів і вмію використовувати їх за призначенням.

CSS

Знаю багато CSS-властивостей і вмію правильно розташовувати контент на сайті, який ви зараз бачите.

JS

За його допомогою можна налаштовувати різні кнопки та інші елементи. Я поки що тільки вчуся.

React

Це фреймворк, який допомагає створювати великі сайти. Поки не почав вчити, але бачив відео, як його використовують.

GitHub

Сайт, де можна викладати свою роботу, а інші користувачі можуть переглядати ваш код і навчатися. Його також використовують для командної роботи. Ви зараз переглядаєте цей сайт на GitHub.

Git

Інструмент для зберігання версій вашого коду. Якщо щось зламалося, можна повернутися до попередньої версії. Поки ще не користувався, але він встановлений на моєму комп'ютері.

Visual Studio Code

Популярний редактор для коду. Мені подобається за швидкість і кастомність. Є багато плагінів, які допомагають прискорити роботу або змінити тему редактора.

Математика

Алгебра

Допомагає працювати з формулами, відсотками та пропорціями, що корисно під час розрахунку розмірів елементів і створення анімацій.

Геометрія

Дає розуміння форм, кутів і відстаней, що допомагає під час роботи з макетами, SVG-графікою та адаптивним дизайном.

Кастомні стилі

Styles

Також я створюю кастомні стилі для масштабних і невеликих сайтів. Один із таких стилів я зробив для Google Classroom. За допомогою цього стилю ви можете перетворити світлу тему Classroom на темну та виправити деякі помилки, допущені розробниками.
